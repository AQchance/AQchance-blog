## 陈奕迅的《任我行》

当一个人听歌时，总能被循环到的这首歌触动。这首歌想表达的主题是关于个体在社会中的独立自由与社会高度集体化之间的矛盾。这首歌的一些歌词的含义隐藏的很深，需要进行一些诠释来辅助理解。我自己在听这首歌许多遍之后，对于这首歌的歌词也有了一些自己的理解，因此在此记下自己的解读以自娱。

## 歌词解析

> 天真得只有你 \
> 令神仙鱼归天要怪谁 \
> 以为留在原地不够遨游 \
> 就让它沙滩里戏水

这样天真的事情也只有你做的出来，令神仙鱼归天了，可是那又能怪得了谁呢？你以为它留在狭小的鱼缸里面不够自由，因此你就让它去沙滩里面戏水游玩。神仙鱼是淡水鱼，尽管到了更加自由更加宽广的大海，可是适应不了环境，这份自由反而成为了它归天的原因。

> 那次得你冒险半夜上山 \
> 争拗中队友不想撑下去 \
> 那时其实尝尽真正自由 \
> 但又感到无趣

那次你和几名队友一起冒险半夜爬山，但是出于对未知和黑夜的恐惧，队友不想继续爬下去了，于是你和队友产生了争拗。也许是因为队友的不坚定，尽管你已经尝到了真正的自由的滋味了，可是总觉得没有什么意思。

> 不要紧，山野都有雾灯 \
> 顽童亦学乖不敢太勇敢 \
> 世上有多少个缤纷乐园 \
> 任你行

不要紧的，山上野外都有雾灯，前路并不是一路黑暗。可是你还是没有勇气走下去，即使是平日里顽皮的自己，也学乖了不敢太勇敢，于是你们只好原路返回，下山去了。在下山的路上你不断安慰自己，世界上有无数个缤纷多彩的乐园可以供自己任意嬉戏，又何必执着于这黑夜中的深山呢？

> 从何时你也学会不要离群 \
> 从何时发觉没有同伴不行 \
> 从何时惋惜蝴蝶困于那桃源 \
> 飞多远有谁会对它操心

不知道从何时起，你也学会了不要离开群体；也不知道从何时起，你发觉到没有同伴是不行的；又不知道从何时起，你总为被困于桃源的蝴蝶惋惜，不管飞到多远都没有谁会为这离群的蝴蝶而关心。

> 曾迷途才怕追不上满街赶路人 \
> 没人理睬如何求生 \
> 顽童大了没那么笨 \
> 可以聚焦于康庄旅途然后同沐浴温泉 \
> 为何在雨伞外独行

就是因为曾经迷路过，所以才会担心、害怕追不上街道上形形色色的路人，要是没有人理睬自己，那简直不敢想象应该如何生存下去！小时候的顽童长大了之后也没有那么笨，当然会选择走在康庄旅途之上，然后和大家一起沐浴温泉。有人群的陪伴，又何必在雨伞外独行呢？

> 这么多好去处 \
> 漫游到独家村去探谁 \
> 既然沿着寻梦之旅出发 \
> 就站出点吸引赞许

世界上有这么多的好去处，漫游到那荒郊野岭的孤独村落里去找谁呢？既然说过了要出发去寻找梦想，那就站出来吸引大家的目光，让大家赞许你吧！

> 逛够几个睡房到达教堂 \
> 仿似一路飞奔七八十岁 \
> 既然沿着情路走到这里 \
> 尽量不要后退

从出生开始，你就在住着形形色色的卧室，经历了几段感情，和不同的人相处一室之后，你发现这些睡房也都大同小异，因此你最终走到了教堂里，就这样，你从一个顽童蜕变成为了大人。然后你发现，你的人生和大家过的一样，没有什么特色，好像只要步入了正轨就开始像火车一般飞奔，眨眼之间就会到七八十岁。不过，既然都已经沿着情路走到了这里了，那就继续往前走吧，尽量不要后退了。

> 亲爱的 闯遍所有路灯 \
> 还是令大家开心要紧 \
> 抱住两厅双套天空海阔 \
> 任你行

亲爱的啊，闯遍了所有的路灯，走过了所有的马路之后，你应该明白了，还是让大家都开开心心才是最重要的。手中的两厅双套几十平米的钢筋混凝土的房子，不也是天空海阔任你行吗？

> 从何时你也学会不要离群 \
> 从何时发觉没有同伴不行 \
> 从何时惋惜蝴蝶困于那桃源 \
> 飞多远有谁会对它操心 \
> 曾迷途才怕追不上满街赶路人 \
> 没人理睬如何求生 \
> 顽童大了没那么笨 \
> 可以聚焦于康庄旅途然后同沐浴温泉 \
> 为何在雨伞外独行

~~依旧是副歌~~

> 亲爱的 等遍所有绿灯 \
> 还是让自己疯一下要紧 \
> 马路戏院商店天空海阔 \
> 任你行

亲爱的啊，你已经循规蹈矩等遍了所有的绿灯了，还是让自己疯一下要紧啊。马路、戏院、商店里，尽管依然受着拘束，不也是天空海阔任你行吗？

> 从何时开始忌讳空山无人 \
> 从何时开始怕遥望星辰 \
> 原来神仙鱼横渡大海会断魂 \
> 听不到世人爱听的福音

不知道从什么时候，你开始担心你想要爬的山上一片孤寂，没有人烟；不知道从什么时候，你开始害怕抬头遥望星辰。原来神仙鱼横渡大海就会断魂，你就像那条神仙鱼一样，如果离开了群居的生活，脱离了充满限制自由的社会，你就会失去活力，再也听不到世人爱听的那些喜讯了。


> 曾迷途才怕追不上满街赶路人 \
> 没人理睬如何求生 \
> 顽童大了没那么笨 \
> 可以聚焦于康庄旅途然后同沐浴温泉 \
> 为何在赤地上独行

就是因为曾经迷路过，所以才会担心、害怕追不上街道上形形色色的路人，要是没有人理睬自己，那简直不敢想象应该如何生存下去！小时候的顽童长大了之后也没有那么笨，当然会选择走在康庄旅途之上，然后和大家一起沐浴温泉。有人群的陪伴，又何必在**赤地上**独行呢？

> 顽童大了别再追问 \
> 可以任我走怎么到头来又随着大队走 \
> 人群是那么像羊群

身为顽童的我已经长大了，就不要再追问我为什么明明可以自由自在的走可是到头来又只能随着大队踏步了。人群是多么像羊群啊！

## 感悟

这首歌听来总有一种宿命论的意思，我们所有的努力都是为了踏上正轨，为了合群，可是一旦踏上了正轨，一旦合了群，那生活似乎又都变成了无聊的重复，成为了毫无特色的复制品。“原来神仙鱼横渡大海会断魂”是全歌最精彩的一句歌词，我们渴望自由，可是又离不开这样高度社会化的生活，我们在狭小的鱼缸里可以“拘束的生存”，可是一旦离开了牢笼一般的鱼缸去往了象征绝对自由的大海，却只能“自由”地死去，我的博客的签名也正是这句歌词。

过年回家之后一定要和好朋友们去KTV再唱一次这首歌～
